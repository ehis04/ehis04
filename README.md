## 📌 Current Projects

### ⚽ xgGaffer

A full-stack football analytics and club management platform built as a final-year capstone project at DCU. The system combines multi-source data pipelines, seven ML prediction engines, an AI-powered query interface, and a suite of club management tools into a single production-quality product.

**Core features:**

- **Seven ML prediction engines,** covering match outcome, scoreline distribution, Elo team strength, season simulation, injury availability, tactical pass success, and starting lineup prediction
- **AI query system,** a deterministic five-layer pipeline combining intent classification, fuzzy entity extraction, per-intent engine handlers, and Claude Haiku as a grounded response narrator
- **Multi-source data pipeline,** aggregating tens of thousands of match and player records across five leagues and twelve seasons; orchestrated via Celery and Redis
- **Club management,** with membership roles, invite/join workflows, fixture scheduling, and resource storage via Cloudinary
- **Tactics board,** an SVG-based animated pitch editor with multi-frame animation and full backend persistence
- **Referee and laws module,** with AI-parsed laws of the game, an interactive quiz system, and structured course progression
- **Social layer,** with posts, follower visibility controls, and activity feeds

### 🏅 Social Olympics

A cross-platform Olympic-style competition platform where groups of friends create and compete in custom events, track medals and points across team and individual leaderboards, and share results through a social layer.
Core features:

52-event library across 16 categories, covering athletics, swimming, racket sports, combat sports, weightlifting, and more, each with configurable weight multipliers and best-of-N result selection
Scoring engine, with a 10/6/3/1/0.5/0.1 points scale, MVP and worst performer bonuses, snake draft team balancing, and a four-stage tiebreaker sequence including sealed event nomination
Competition lifecycle management, with host controls, co-host roles, join/invite workflows, result submission, a dispute window with evidence uploads, and animated podium reveal
Social layer, with a real-time activity feed, group chat, direct messaging, notifications, and competition history across career statistics
Moderation and compliance, with a report system, keyword filtering, admin review queue, and full GDPR compliance for a 16+ user base

### 2. AI Observability & Prompt Audit Platform

A developer tool for teams building LLM-powered products, addressing a gap that has emerged as AI integration becomes standard: the lack of structured visibility into how prompts, model outputs, and downstream behaviour evolve over time.

Production LLM systems are difficult to debug, audit, and improve systematically. Teams currently rely on ad hoc logging, manual spot-checking, and scattered evaluation scripts. This platform would centralise that workflow.

**Planned features:**
- Prompt version tracking with structured diffing and rollback
- Automated evaluation runs comparing outputs across prompt versions against defined criteria
- Regression detection; flagging when model behaviour changes after a prompt or model update
- Token cost and latency analytics per prompt, workflow, and model
- Team-level audit trails for compliance and accountability in regulated industries
